# UI revamp — archived prototypes

Snapshots of design exploration from the `ui-revamp` arc, kept for future
reference if/when we revisit the visual language. Nothing here is wired into
the running app.

- `ui_ideas/` — HTML variants and review PNGs from the first round of
  prototypes (Autobahn, Datacore, LedWall, Nixie, Phosphor, Synthwave,
  Walkman, etc.) plus `costs.md` notes.
- `ui_structure_rethinking/` — second round, exploring shell structure
  (`v1_monk.html`, `showcase.html`).
- `icons/` — `gallery.html` icon explorations.

Last touched May 2026, end of the `ui-revamp` branch. Open the HTML files in
a browser to view; PNGs render directly. The QA arc that produced these
(`bugs.md`, `qa_report.md`, `plan.md`, `intent.md`) is preserved in branch
history — see the `ui-revamp` branch on the remote, or `git log -- _ui_revamp`.
